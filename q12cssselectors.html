Question 12: CSS Selectors

CSS Selectors are used to target HTML elements to style them. There are different types of selectors like ID, Class, and Attribute selectors.

ID Selector

This heading is styled using an ID selector (#heading).

Class Selector

This paragraph is styled using a class selector (.para).

Attribute Selector

The input and button above are styled using attribute selectors like input[type="text"] and button.
