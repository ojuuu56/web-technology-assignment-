Question 11: Different Types of CSS

CSS (Cascading Style Sheets) is used to style HTML elements. It can change colors, fonts, spacing, and layout of web pages.

1. Inline CSS

Inline CSS is applied directly to an HTML element using the style attribute.

2. Internal CSS

Internal CSS is written inside the <style> tag in the <head> section of an HTML page.

3. External CSS

External CSS is stored in a separate .css file and linked to the HTML using the <link> tag.
